O que é um ataque de Cross Site Request Forgery? Tipo de ataque informático malicioso a um website no qual comandos não autorizados por um utilizador são transmitidos inadvertidamente por este. O ataque explora a confiança que um website tem no browser.
HTTP vs HTTPS O HTTP (HyperText Transfer Protocol) permite a comunicação entre sistemas e é o protocolo mais usado para transferir dados de servidores para os browsers dos utilizadores. Porém, existe um problema com este protocolo pois a informação que circula por meio do mesmo não está cifrada. A solução passa por usar o HTTPS (HyperText Transfer Protocol Secure) que cria uma conexão cifrada segura entre o servidor e o browser.
